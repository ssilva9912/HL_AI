# Personal Homelab AI project

A modular local AI platform for offline document search, personal knowledge retrieval, media indexing, and homelab/network integration.

## Goals

- Run local LLMs on personal hardware
- search personal files, notes, PDFs, and code
- keep core functionality usable offline
- Build modular components that can move from laptop to server later on.
- learn AI infrastructure, Docker, networking, Linux, and cybersecurity.

## Initial Modules

1. Local LLM interface
2. File ingestion
3. Text parsing
4. Chunking
5. Embeddings
6. Vector database
7. Retrieval-augmented generation
8. Web UI
9. Network integration
10. Security hardening
